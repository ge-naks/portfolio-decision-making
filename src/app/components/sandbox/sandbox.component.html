Current harvest: {{harvest}}

Here we have a predefined ceiling. This ceiling represents a maximum allowed value for a given portfolio. We assume that the growth follows brownian motion with drift. Each point is generated procedurally, and any wealth generated above the ceiling is "harvested", taken aside to be utilized in other investments, expenses etc. A button below is provided to generate several different simulations. Notivce how the harvest can change from simulation to simulation. In fact, you might get unlucky and bankrupt your portfolio quickly, losing out on earnings, or earning nothing entirely. Think how you would make these decisions!
